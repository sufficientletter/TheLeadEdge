# Logging Rules

- All significant actions must be logged.
- Session logs go to `./logs/`.
- Team Lead journal entries go to `TeamLeadJournal/`.
- Log format: timestamp, agent, action, outcome.
- Journal entries: `YYYY-MM-DD_thoughts.md` with session goals, decisions, outcomes.
- NEVER log PII (client names, phone numbers, email addresses, home addresses).
- Sanitize all log output to remove personally identifiable information.
